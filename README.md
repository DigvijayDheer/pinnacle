# Pinnacle Next.js

A visually rich Next.js project showcasing a premium UI experience with animated components, responsive sections, and a curated design system.

## Features

- Built with Next.js 15 and React 18
- Tailwind CSS for styling
- Radix UI primitives for accessible components
- MUI icons and rich animation support
- Modular page and component structure
- Includes custom UI utilities and interactive experience sections

## Project Structure

- `src/app/` — main app pages and layout
- `src/components/` — core landing page sections and UI pieces
- `src/components/ui/` — shared UI components and utilities
- `src/styles/` — global project styles

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

Run linting:

```bash
npm run lint
```

## Notes

- This project is configured as a private Next.js app.
- The UI is designed for a premium or enterprise-facing landing experience.
- Inspect `src/components` to explore reusable sections and design patterns.
